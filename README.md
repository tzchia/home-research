## 撰寫原則

1. **第三方認證優先**：NSF / AHAM / Allergy UK / ECARF / SGS / TTRI > 廠商廣告數字
2. **5 年總持有成本**：機器只是入門票，耗材才是長期支出
3. **使用情境決策樹**：不給單一推薦，按家庭條件給對應選項
4. **下手前自我檢核**：採購陷阱與廠商不會告訴你的細節
