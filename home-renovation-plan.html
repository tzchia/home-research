新家裝潢規劃

7 樓華廈 · 第 6 樓單層一戶　|　更新：2026-06-29

以「不打牆」為前提，把現有 3 房 2 廳雙衛的格局，對應到「夫妻 + 兩個寶寶（未來可能 + 岳父）」的人口結構，做機能定位與設計策略。重點在用櫃體、軟裝、燈光分區，而非動隔間 — 省錢、零結構風險、工期短。

一句話結論

3 房 2 廳 · 雙衛 室內 28 坪 預算 100 萬 天花 2.7m / 樑下 2.3m 不打牆 · 系統櫃為主

現況格局本來就很好用 — 三房剛好對上「夫妻 + 兩寶」，又是雙衛浴（一套貼主臥、一套公共），早上不搶廁所。決定不打牆是聰明的：省下拆除/清運/泥作復原費、零結構風險、不必煩惱哪道是剪力牆。

裝潢重心因此轉到 3 件事：① 用櫃體與軟裝把每個區的個性做出來；② 保住 2.7m 淨高的爽度、只「化掉」那幾支樑；③ 房間分配預留人口成長彈性，但不為「5 年後可能」現在就硬花錢。

3淨高策略：保高，只「化掉」樑

2.7m 淨高其實比一般公寓（2.4～2.6m）寬敞，是這間房子的隱形優勢 — 不要為了藏樑把它砍掉。

核心原則：不要為了藏樑把整片天花拉平到 2.3m。那等於把 40cm 的高度優勢全砍掉、整屋變壓迫。正確做法是「保高 + 只處理樑」。
🪟

天花留高

客餐廳走極淺天花或裸天花，留住 2.7m 的開闊感；別整片包平。

💡

樑側間照

樑兩側做間接燈帶或圓弧導角，把樑「化掉」而非「包掉」。樑是 RC 沒法埋嵌燈，要在樑旁配光。

🔆

薄型燈具

選軌道燈 / 吸頂薄燈，避免吊燈再吃高度。

🗄️

高櫃避樑

衣櫃 / 高櫃到頂可做 ≈ 2.6m 收納拉滿，但櫃體與上掀床要先對齊樑位，別卡一支樑在櫃子中間。

❄️

冷氣走樑下

冷媒管 / 排水走樑下時先算好洩水坡度，出風口別被樑擋。

🚪

樑下=走道

樑下 2.3m 拿來當走道、過門完全沒問題（門高才 2.0～2.1m），把矮的地方放在不影響體感的位置。

7預算分配：100 萬 / 28 坪的現實

先講真話 — 這個預算的合理打法，跟「滿屋系統櫃」是有張力的。

100 萬 ÷ 28 坪 ≈ 3.6 萬 / 坪，屬於「重點式輕裝修」級距（滿屋木作豪裝要 6～10 萬 / 坪）。好消息：你「不打牆 + 系統櫃為主」剛好就是這個級距的最佳打法，錢花在刀口上。
⚠️ 本頁預算假設：這 100 萬是「裝潢工程款」，不含冷氣主機（分離式 ×4 約 15～20 萬，建議獨立編列）與活動家具家電（冰箱 / 洗衣機 / 沙發 / 床 / 餐桌）。若 100 萬要「全包」，工程款會被壓到 ~65 萬，本頁所有金額需再下修一級。這個假設對不對，請先跟我確認。

8系統櫃：做少 / 做中 / 做多

你要的三版本。核心取捨：系統櫃做越多，基礎工程或總預算就得讓步。

系統櫃概略行情：到頂衣櫃（約 8 尺寬）≈ 3～5 萬 / 座、玄關鞋櫃 ≈ 2～4 萬、電視櫃牆 ≈ 3～6 萬、餐邊 / 電器櫃 ≈ 2～5 萬、書桌書牆 ≈ 2～4 萬。板材（E1 / EGGER）、五金（Blum）等級會上下浮動。
版本系統櫃預算包含項目取捨 / 總價影響
做少 ≈ 15 萬 主臥衣櫃 + 兩間次臥衣櫃 + 玄關鞋櫃（純必要收納） 剛好落在 100 萬內 ✓ 客餐廳用活動家具，留白多、未來彈性最大
做中 ≈ 32 萬 做少版 + 媒體櫃 / 布幕牆（投影） + 餐邊 / 電器櫃 + 主臥化妝 / 書桌 + 一間小孩書桌櫃 需 加到 ~116 萬，或砍基礎（地板沿用舊的 / 廚衛降一階）回壓到 100 萬
做多 ≈ 55 萬 全室到頂櫃 + 布幕牆 + 餐櫃 + 書牆 + 客廳臥榻 + 陽台收納 + 浴櫃 100 萬內幾乎不可能，需加到 ~135 萬，或大砍基礎（不建議犧牲水電 / 防水）

9電器盤點 → 插座 / 迴路規劃

你問的「要先確認電器才能決定插座」— 完全正確，這是裝潢最常被做反的順序。

正確流程：① 先盤點電器（現有 + 想買）→ ② 標出每台的位置 / 電壓 / 是否要專迴 / 要不要給排水 → ③ 最後才定插座數量、位置、迴路、弱電。
口訣：插座寧多勿少 — 事前補一個插座 ≈ 幾百元；事後拉線要打牆補土重漆，貴 10 倍還毀裝潢。
✅ 你們已定的電器決策（2026-06-29）：① 廚房用瓦斯爐（免 IH 220V 專迴）② 冷氣4 室內機 / 2 室外機（主臥+客廳共用一台、兩次臥共用一台）③ 投影機取代電視 ④ 雙衛各裝三合一暖風機 ⑤ 後陽台洗脫烘評估中。下表已套用這些決策。
🔴 你們這套組合，這些「專迴」一定要事前拉（事後幾乎補不了）： 冷氣 ×2 迴（一對二是「每台室外機一條 220V 專迴」，不是 4 條）、雙衛三合一暖風機 ×2 迴（高功率，不可與免治/吹風機共迴）、後陽台洗脫烘預留 1 迴（評估中也先拉）。
瓦斯爐已排除 IH，省下一條 220V — 但烤箱/蒸烤箱/洗碗機若要，仍各需專迴，請先決定。
📽️ 投影機 3 個一定要先處理（這是投影取代電視的關鍵）：
① 避樑！客廳上緣有大樑（第 2、3 區）— 天花吊架位置與投影路徑都要閃過樑，否則畫面被切或裝不上。
② 預埋訊號管：從天花投影機位 → 媒體櫃 / 布幕牆，要預留穿線管（HDMI 會改版，留「空管」日後好換線），加投影機電源 + 布幕牆電源（電動布幕）。
③ 遮光：客廳是全屋最亮的採光帶 — 投影怕光，窗簾要做全遮光，或搭抗光（ALR）布幕 + 高流明機種，白天才看得清楚。
❄️ 一對二冷氣補充：主臥+客廳共用一台室外機是合理配對（兩者使用時段多半錯開：客廳白天晚間、主臥深夜）。但要請廠商確認 ① 組合容量夠不夠兩房傍晚同開；② 2 台室外機放置位置（冷氣板 / 後陽台，避免互吹熱風）；③ 冷媒管走樑下時的洩水坡度與長度限制。

💡 洗脫烘「評估中」也要先預留：不管最後買不買，進排水 + 220V 迴路 + 排風 / 散熱空間現在留好幾乎不花錢；之後想裝才不用打牆。洗脫烘（單機洗+烘）省空間但烘衣較慢、5 年總成本與「獨立洗 + 熱泵烘」要比 — 這題之後可幫你開一份家電比較。

弱電 / 網路 / 預留（別漏）

  • 投影機↔布幕牆穿線空管 — 預留管徑足夠（HDMI / 光纖 HDMI 日後好抽換），別直接埋死線。
  • 弱電箱位置先定 — router 放這、各房網路孔回拉到這（每房至少 1 個有線網路孔，投影 4K 串流更穩）。
  • Wi-Fi AP 佈點 — 28 坪長型格局，客廳 + 主臥側各留一個 AP 電源 + 網路孔，避免訊號死角。
  • 嬰兒 / 小孩房多留床邊插座（監視器、夜燈、加濕器、未來檯燈書桌）。
  • 玄關留插座（電子鎖備援、烘鞋機、掃地機可改放這）。
  • 餐桌邊留插座（火鍋、氣炸鍋、桌上小家電）— 超常被忘記。
  • 配電箱總容量要算夠 — 冷氣 ×2 + 暖風 ×2 + 洗脫烘 220V 同時開別跳電（瓦斯爐已省一條 220V）。

10採購自我檢核：別被坪數灌水

裝潢報價務必問清楚：設計師是用「室內 28 坪」還是「權狀 32.41 坪」抓單價？ 有些報價故意用大坪數 × 單價，總價就膨脹。你家天花 / 地板 / 油漆的實際施作面積是跟室內淨坪走的 — 用 32.41 報價等於多算你 4 坪的錢。要求對方明列計價基準。
檢核項目為什麼重要
計價基準是室內坪還是權狀坪？差 4.4 坪，用大的報價直接膨脹總價。
哪幾道是剪力牆 / 結構牆？就算不打牆也要知道 — 釘櫃、開孔、走管都受限。請設計師標在圖上。
樑位有沒有疊到平面配置圖上？高櫃 / 上掀床 / 冷氣出風口都要避樑，沒標清楚會現場踢鐵板。
系統櫃 vs 木作的比例與報價？同收納量價差很大，要分項看，不要包成一筆。
5 年總持有成本（含家電耗材 / 保固）裝潢只是入門票，後續維護才是長期支出 — 一貫要看 5 年。

11還缺這些，就能定案開工

預算（100 萬）與系統櫃版本方向已定。剩下幾項補齊，就能進到設計師畫施工圖階段。

  • 確認預算範圍 — 100 萬是否為「純工程款」（不含冷氣主機 + 活動家具家電）？這決定本頁所有金額是否要下修。
  • 填電器盤點表（第 9 區）— 跟太太一起把現有 + 想買的電器逐台填上位置 / 電壓，水電圖才畫得準。這是現在最該做的一件事。
  • 系統櫃版本拍板 — 建議「做少打底 + 做中分期」；若要一次做中 / 做多，需確認預算是否上修。
  • 不可動的牆 — 請設計師指出哪道是剪力牆 / 結構牆（華廈通常 1～2 道），標到平面圖上。
  • 風格 — 目前沒想法 OK；可先收集 3～5 張喜歡的實景圖（Pinterest / IG）給設計師抓方向，比講形容詞準。
